# Whaleleap Tech website

Static, Vercel-ready website reconstructed from the public Whaleleap Tech content and strengthened for SEO/GEO.

## Safe deployment workflow
1. Upload all contents of this folder to the connected GitHub repository.
2. Review the generated Vercel Preview deployment before assigning the production domain.
3. Confirm all product specifications, claims, documents and contact details before production launch.
4. Do not delete the current production deployment until the preview is approved.

## Included
- 17 indexable pages
- Product, Article, Organization and WebPage structured data
- robots.txt, sitemap.xml and llms.txt
- Canonical URLs and social metadata
- Responsive layout and accessible semantic HTML

## Important
This reconstruction deliberately avoids unverified certification, export-country, capacity and efficacy claims. Add those only with current supporting evidence.
